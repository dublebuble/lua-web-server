Message from web server

If you can see this page it means that the web server is running as expected.
This web server was written in the Lua programming language by Samuel Saint-Pettersen
and uses LuaSocket.

To kill the web server:

  1. Open up a shell prompt and navigate to the directory containing webserver.lua
  2. Run the telin script
  3. Once connected in telnet, enter kill
Alernatively, type telnet localhost 8080 and once connected enter kill
